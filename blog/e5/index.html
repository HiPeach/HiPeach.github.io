教程 | E5，从入门到入坟

发表于2022-04-09|更新于2022-04-09|教程
|字数总计:1.2k|阅读时长:3分钟|阅读量:

前言
E5，全称Microsoft 365 E5 开发人员计划
即微软为开发者人员准备的专属账号
账号属于微软账号分类中的工作或学校账户
而且，E5从注册到使用全程免费，并且支持永久自动续订
E5具体优势有许多，以下为主要内容

  • 免费的 Microsoft 365 E5 开发人员订阅 [免费正版Office全套应用]
  • 默认为1T实际可扩容至每个账号5TB的OneDrive存储空间 [几乎∞的存储空间]
  • 一共可以分配25个子账号，并且全程免费 [这辈子都够用了]
    欢迎阅读《E5，从入门到入坟》

注册

通过电脑打开开发人员计划申请页面
登录自己的微软账号，必须为个人账户，不可为工作或学校账户
如图1.1，选择国家/地区为中国，公司名称可以随意填写，语言更改为中文简体并同意第一条条款

如图1.2，选择个人项目

如图1.3，选择Office加载项与Outlook

如图1.4，沙盒选择可配置沙盒

如图1.5，创建用户名和域，并设置密码，安全性为ProMaxUltraFold

如图1.6，设置手机号码并接收验证码，支持中国大陆+86
注意：按下发送代码后，将调用Captcha，如果你按下按钮没有反应，请使用Magic

至此，已经完成注册的步骤，你将会在仪表盘看到剩余天数

设置账户&扩充OneDrive

首先，无论是给你自己还是给他人使用，你都需要分配账户
打开账号管理界面，使用账户用户名@域.onmicrosoft.com登录
点击添加用户，按照步骤操作，记得分配许可证，不然Ta啥都做不了！
你也可以单击某位用户-OneDrive-已使用的存储空间-编辑-更改为5120GB并保存
至此，你就有了5TB的OneDrive存储空间
另外，E5账户会在若干天后提醒你，设置2FA [微软两步验证]
如果你是电脑用户，或者不方便使用2FA的，请一定不要开启！
尤其是主账户（或任何具有完全管理权限的账户），不然你的E5有101%的概率废掉！
你可以通过添加并且验证手机号来规避2FA
这十分重要！

自动续期

E5只有将近三个月的使用期限，期限到后，若要继续使用，需要活跃地使用开发人员功能
我在这隆重地介绍一个软件——Microsoft 365 E5 Renew Plus
原帖来自CSDN
主要功能：

  • 支持开机自启动后台调用 使用简单方便，无需购买服务器部署，本地应用程序即开即用
  • 两种可选的调用权限 用户未登录作为守护程序调用(需要客户端密码)、程序以登录用户身份直接调用(需要账户密码)
  • 自动配置API权限 登录调用的API权限支持由程序自动配置
  • API种类齐全 43个可选的Microsoft Graph REST API Beta中的API(未来可能会继续添加)
  • 完全随机的API选用模式 从已选定的API序列中随机抽取一个或几个进行调用（个数和API随机）
  • 完全随机的API调用顺序 每轮API的调用先后顺序随机
  • 完全随机的API内容(仅部分API支持) POST写类型的API的上传内容随机
  • 完全随机的API调用时间间隔 随机区段500s-86400s自定义
  • 每日工作时间自定义 在日常工作时间进行调用，适用于从不关机重启的服务器设备
  • 每周工作日自定义 在每周的工作日进行调用，适用于从不关机重启的服务器设备
  • 运行智能暂停 防止因持续在错误配置下运行而导致的账户封禁
  • 运行配置自动保存 可保存运行配置，无需反复设置
  • 无限制账号个数 理论上允许无限制个数账号后台自动调用
  • 支持自定义客户端密钥 允许非登录调用使用自定义密钥
  • 自定义美化界面 支持自定义背景图片，半透明度以及毛玻璃效果

如有需要，请去CSDN原帖查看（懒得搬了）
